Course Information

Code and Type: SlCS343-Essential Course
Numbers Of Credits 5
Pre-requisites: Programming 1
Duration: 2 months
Description: This course provides an introduction to Object Oriented Programming concepts using Python. The course emphasis is on the on the object orientated facilities of Python and how they can be used to create modular and re-usable code. This course is to develop programming skills of students, using functions, pointers, string and object oriented programming concepts, learn the concept of class and object using C++ and develop classes for simple and complex applications.
Rating:
Pricing: 10$/credit = 50 $

Learning outcomes

LO#1: Understand the concepts of functions, pointers and classes and Recognize the concepts of object-oriented programming and basic structure of python programming.
LO#2: Apply the principles of functions and pointers and the concepts of object-oriented design such as constructor/destructor, encapsulation, polymorphism and inheritance.
LO#3: Design and construct python programs, by using various inheritance concepts and virtual function for given problem.
LO#4: Analyze and evaluate computer programs using the C++ programming language.

Course Outline

Week Topic
1 Revision of Programming I: input/output, conditional statements, loops and arrays
2 Revision of Programming I: input/output, conditional statements, loops and arrays
3 Functions: pass-by-value , pass-by-reference, passing arrays
4 String : how to deal with string and its functions (assignment 1)
5 Concepts of OOP: Introduction OOP, Procedural Vs. Object Oriented Programming, Program structure, namespace, objects and classes (quiz 1)
6 String : how to deal with string and its functions (assignment 1)
7 Introduction to classes _ access specifies _ private, protected and public data members, Attributes and methods
8 Constructors _ default constructor _ Parameterized constructors _ destructors, static data and static methods - Inheritance -Overloading
